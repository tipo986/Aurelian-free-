# Aurelian-free-
Client for Minecraft

Aurelianfree (Free Version)

Aurelianfree  – это бесплатный клиент для Minecraft 1.16.5, созданный для улучшения игрового опыта. Проект находится на начальной стадии разработки, но уже включает базовые и полезные функции для комфортной игры. Мы стремимся сделать клиент доступным, стабильным и удобным. Клиент будет поддерживать версионирование – следите за обновлениями

🌟 Особенности (Free Version)

· KillAura (LegitAim) – интеллектуальный ассистент прицеливания с настройками:
  · Случайное отклонение головы (Yaw/Pitch)
  · Задержка смены направления взгляда
  · Режимы рандомизации: плавный, рывками, случайные точки
  · Опции: критические удары, ломать щит, атака в воде и др.
· AutoPotion – автоматическое кидание зелий (в разработке, появится в следующих версиях)
· Визуальные улучшения – планируются в будущих версиях
· Оптимизация – снижение нагрузки на систему

Все модули легко настраиваются через удобное внутриигровое меню (по умолчанию RShift).

📥 Установка

1. Скачайте последнюю версию AurelianDLC-free из релизов (выберите актуальную версию).
2. Поместите файл клиента в папку версий (например minecraft/version).
3. Запустите Minecraft 1.16.5 с установленным Aurelian free.

🎮 Использование

· Открытие меню: нажмите RShift
· Включение модулей: выберите нужный модуль в списке и переключите его.
· Настройка параметров: для каждого модуля доступны индивидуальные настройки (дистанция, FOV, рандомизация и т. д.).
· KillAura: выберите режим LegitAim, настройте случайные отклонения головы под свой стиль игры. Опция «Атака в воде» позволит атаковать в воде с замедлением и дополнительной задержкой.
